### 功能描述
1. 从单词列表中随机生成单词；
2. 输入正确单词后累计分数并根据难度设置增加相应的剩余时间；
3. 难易程度记录在LocalStorage中；
4. 游戏结束后显示分数并提示是否继续游戏；
